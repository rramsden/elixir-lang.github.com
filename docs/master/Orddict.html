Orddict

This module implements a dictionary based that stores items as a list of tuples. It is a simple wrapper around Erlang's orddict module and exposed via the Dict module.

Check the Dict module for examples and documentation.

new(pairs)

Creates a new dict from a list of pairs.

Examples

Orddict.new [{:b,1},{:a,2}]
#=> [a: 1, b: 2]

new(list, transform)

Creates a new dict from a list of elements with the help of the transformation function.

Examples

Orddict.new ["a", "b"], fn x -> {x, x} end
#=> ["a": "a", "b": "b"]
